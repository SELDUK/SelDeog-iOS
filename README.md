## ⭐️ 개요
셀덕은 셀프 덕질의 줄임말 <br>
타인을 조건없이 아껴주는 소위 '덕질' 문화가 발달한 현대 사회에서, <br>
스스로를 사랑하지 못하여 상처받는 사람들을 위하여 부담없이 자기애를 형성할 수 있는 앱 <br>
자기 자신을 덕질하며 새로운 자신의 모습을 발견하고 더욱 사랑할 수 있는 계기를 만들어 줌
